Create "Optimus Projects" Flyer in Canva — Quick Guide

Goal
- Recreate the flyer exactly in Canva (A4 portrait) and export as PNG.

Quick setup
- New design → Custom size: 210 mm × 297 mm (A4, Portrait).
- Upload assets: `OPTIMUS_LOGO.png`, `project.png`, `team.png`, `whatsapp.png`, `image.png` (QRs).

Fonts
- Headline / display: Montserrat (weights 700/800)
- Body / UI: Inter (400/600/700)
- If Montserrat/Inter not available, use similar (Poppins / Roboto).

Canvas grid & units
- Use a 12-column grid or guides. Keep 20–30 mm page margins for safe area.

Exact Colors (use these HEXs)
- Deep Navy (header dark): #0F172A
- Slate Navy (header mid): #1E293B
- Mid Slate (text / accents): #334155
- Gold Bright: #F59E0B
- Gold Dark: #D4A017
- Light Gray (borders/soft backgrounds): #E5E7EB
- Off White / Page BG: #F1F5F9
- Pure White: #FFFFFF

How to build (concise steps)
1. Page background
   - Add rectangle covering full page.
   - Fill: solid `#F1F5F9` (or subtle vertical gradient from `#FFFFFF` → `#F8FAFC`).

2. Header block
   - Add rectangle across top ~28% of page height.
   - Fill: linear gradient from `#0F172A` to `#1E293B` at ~135°.
   - Add logo: Upload `OPTIMUS_LOGO.png`. Place top-right, size ≈ 20 mm wide, margin ~6–8 mm from top/right.
   - Left side: Add text "OPTIMUS PROJECTS" using Montserrat, small tagline below in uppercase with color `#D4A017`.
   - Big headline: place centered-left, use Montserrat bold, set highlight word color gradient using `#F59E0B` → `#D4A017` if Canva supports; otherwise color the word `#F59E0B`.

3. Wave divider
   - Create thin rectangle under header and add another rectangle with gold gradient (`#F59E0B` → `#D4A017`). Rotate/clip to create angled wave.

4. Main content columns
   - Create a two-column layout (Left ~60% width, Right ~40% width).
   - Left column: "Our Comprehensive Services" title (Montserrat), then 2-column grid of service boxes.
     - Service box style: white background, border `#E5E7EB`, rounded corners, small gold circular check `#F59E0B`.
   - "Why Choose Optimus" card: dark rounded rectangle filled `#0F172A`→`#1E293B` with gold title `#F59E0B` and white text.

5. Right column
   - Top image frame: upload `project.png`, place inside rounded frame with a gold 3px outer border (use two rectangles: outer gold, inner mask).
   - Second image: `team.png` similarly styled.
   - CTA box: rounded rectangle filled `#D4A017` → `#F59E0B`, text centered, button rectangle `#0F172A` with gold text.

6. Footer
   - Full-width dark bar (same header gradient).
   - Left: contact items with small gold circular icons.
   - Right: two QR placeholders; place uploaded QR images and add small gold captions.

Spacing & sizes (guideline)
- Page margin: 20–25 mm
- Header height: ~80 mm (≈28% of 297 mm)
- Logo width: ~20 mm
- Headline font size: scale until visually similar (use preview zoom)
- Service box height: ~18–22 mm each

Export
- File → Download → PNG
- For print: set Size 300 DPI (or use PDF print with 3 mm bleed).

Tips
- Use grouped elements for repeating components (service box, image frame).
- If Canva gradient options are limited, make gradients as separate images in a simple editor, then upload.

That's it — you can follow these steps in Canva and match the colors exactly using the HEX values above.
